# EasyStat: Automated Statistical Analysis, Visualization and Multi-Format Narrative Reporting

EasyStat provides a unified pipeline for automated statistical analysis
and plain-language narrative reporting in R. The four-step pipeline —
Core Statistical Engine, Metric Extractor (broom), Narrative Generator
Module, and Unified Result Object — transforms raw data into
publication-ready output with a single function call.

[`easy_describe`](https://itsmdivakaran.github.io/Easystat/reference/easy_describe.md),
